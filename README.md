# Laundry Parts Hub - نظام إدارة قطع غيار المغاسل الصناعية

منصة متكاملة وذكية لإدارة تجارة قطع غيار وآليات المغاسل الصناعية، مدعومة بالذكاء الاصطناعي والأتمتة المتقدمة.

## الميزات الرئيسية

### 1. كتالوج شامل للمنتجات
- قاعدة بيانات متقدمة تضم آليات المغاسل الصناعية وقطع الغيار
- بحث ذكي عن طريق الاسم والأعراض والماركات المتوافقة
- معلومات تفصيلية عن كل منتج مع الأعطال الشائعة

### 2. لوحة تحكم مركزية (Dashboard)
- إحصائيات المبيعات والمخزون والطلبات في الوقت الفعلي
- رسوم بيانية وتحليلات متقدمة
- إدارة المنتجات والموردين والعملاء

### 3. نظام إدارة الطلبات
- طلبات عروض أسعار منظمة
- تتبع حالة الطلبات
- إدارة الفواتير والدفعات

### 4. نظام إدارة العملاء (CRM)
- قائمة شاملة للعملاء المحتملين والحاليين
- تتبع التواصل والتفاعلات
- درجات التفاعل والأولويات

### 5. وكلاء ذكية (AI Agents)
- **وكيل التوريد:** مراقبة الموردين وتحديث الأسعار تلقائياً
- **وكيل خدمة العملاء:** الرد الآلي على الاستفسارات والتعرف على صور القطع التالفة
- **وكيل التواصل:** إرسال رسائل موجهة عبر WhatsApp و LinkedIn و Facebook

### 6. نظام الإشعارات الذكية
- إشعارات فورية عند وصول طلبات جديدة
- تنبيهات انخفاض المخزون
- فرص ربحية جديدة من وكيل التوريد

### 7. الاستعلام بلغة طبيعية
- واجهة للاستعلام عن قاعدة البيانات باستخدام اللغة الطبيعية
- تحليل ذكي للبيانات والإحصائيات

## البنية التقنية

### Frontend
- Next.js 14 + React 18
- Tailwind CSS + Radix UI
- tRPC + React Query

### Backend
- Express.js API
- MySQL Database
- Drizzle ORM

## التثبيت

```bash
git clone https://github.com/yourusername/laundry-parts-hub.git
cd laundry-parts-hub
npm install
npm run dev
```
